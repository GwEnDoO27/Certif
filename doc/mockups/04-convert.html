🔍 Éléments de design clés
  • Accent métier bleu — la page utilise blue-500/blue-600 (et non primary indigo) pour la cohérence avec Excel/factures
  • Stepper 2 étapes — Configuration (Settings icon) → Traitement (FileCheck icon), couleur active gradient bleu + scale 1.1
  • Accordéons — 3 sections de codes (généraux, comptables, journal), chevron rotation 90° à l'ouverture
  • Labels flottants — composant InputField custom avec label en absolute top-3, bordure bleue + pulse dot si modifié
  • Drop zone — drag & drop natif + click, accept=".txt" multiple, état actif (border bleu + bg-blue-50)
  • CTA conversion — bouton vert from-green-500 to-emerald-600 pour différencier "action métier" de "sauvegarde config"
  • Téléchargement direct — Blob → download anchor, nom factures_MB_YYYYMMDD.xlsx
  • Analytics — hook useAnalytics('facture-mb') traque l'usage de l'outil

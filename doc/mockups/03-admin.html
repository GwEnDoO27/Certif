🔍 Éléments de design clés
  • Garde route admin — vérif verify-admin au mount, redirige vers / si non-admin
  • Segment control — 3 onglets (Utilisateurs · Applications · Analytics) avec état actif bg-white shadow-sm
  • Stats temps réel — compteurs animés (Framer Motion, delays échelonnés 0.1/0.2/0.3s)
  • CRUD utilisateurs — modal Headless UI pour confirmation suppression, popup d'édition séparé
  • Panel WebSocket — utilisateurs en ligne avec route actuelle, badge vert pulsé (UsersPanel)
  • Badges de rôle — 6 rôles colorés : Admin (indigo), Comptable (vert), Social (orange), Auditeur (jaune), Dev, Client
  • Toast feedback — react-hot-toast pour succès/erreurs CRUD
